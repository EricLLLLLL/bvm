# Initial Concept
BVM (Bun Version Manager) 是一个使用 Bun 开发、专为 Bun 设计的原生、原子化且高性能的版本管理器。它旨在解决传统版本管理器带来的终端启动延迟和环境隔离弱的问题，提供极速的切换体验。

# 项目愿景
为 Bun 生态系统提供最快、最稳固的环境编排工具。

# 核心功能
- **极速终端启动**：采用基于 Shim 的设计，最大程度减少 Shell 启动时的加载开销。
- **原子化私有运行时 (Bunker Architecture)**：维护一个独立于被管理版本的 BVM 运行环境，确保即使卸载所有 Bun 版本，管理器依然可用。
- **混合路径路由**：兼顾全局模式的 OS 原生性能（物理软链接）和项目模式的灵活性（递归搜索 .bvmrc）。
- **环境级原子隔离**：通过直接注入 `BUN_INSTALL`，确保每个版本的全局包物理隔离，杜绝冲突。
- **全球化高速分发**：基于 IP 地理位置与实时竞速策略（Race Strategy），智能选择官方、淘宝或腾讯镜像源，显著提升全球下载速度。
- **智能自愈升级**：基于组件指纹的增量升级与原子替换机制，配合异步版本更新检查，确保管理器始终处于最优状态。
- **智能工具链对齐**：自动处理 `yarn`、`npm`、`pnpm` 等兼容性链接。

# 目标用户
追求极致性能、需要干净的环境隔离体验的 Bun 开发者。

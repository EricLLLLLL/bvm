## One-Line Installation

### macOS / Linux (Recommended)
```bash
curl -fsSL https://bvm-core.pages.dev/install | bash
```

### Windows (Recommended)
```powershell
irm https://bvm-core.pages.dev/install | iex
```

### NPM (Optional)
```bash
npm install -g bvm-core@latest
```

# Welcome from npm

You’re here because you found `bvm-core` on npm. This page is the shortest path to:

- Verify your install
- Get your shell set up correctly
- Subscribe to updates (WeChat)

## 1) Verify BVM is working

```bash
bvm --version
```

## 2) Make Bun resolve through BVM shims

```bash
bvm setup
```

Then restart your terminal and confirm:

```bash
which bun
```
